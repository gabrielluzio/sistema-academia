SISTEMA DE ACADEMIA

Sistema de cadastro e gerenciamento de alunos de academia desenvolvido em Python e MySQL.

Funcionalidades
Cadastro de alunos
Consulta de alunos cadastrados
Atualização de informações
Exclusão de registros
Gerenciamento de planos
Integração com banco de dados MySQL
Tecnologias Utilizadas
Python
MySQL
MySQL Connector
Programação Orientada a Objetos (POO)
Estrutura do Projeto
academia/
├── cadastro_aluno.py
├── main.py
├── .gitignore
└── README.md
Como Executar
Clone o repositório:
git clone https://github.com/gabrielluzio/sistema-academia.git
Instale as dependências:
pip install mysql-connector-python
Configure o banco de dados MySQL.
Execute o projeto:
python main.py
Objetivo

Este projeto foi desenvolvido para praticar conceitos de desenvolvimento back-end com Python, integração com banco de dados MySQL, Programação Orientada a Objetos e operações CRUD.
